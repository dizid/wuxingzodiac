## Part 3: Configure Midjourney Settings

Before generating any artwork, set up these settings once.

1. Go to **midjourney.com/imagine** (this is the main image generator)

2. Look for the settings panel (gear icon or settings area below the text box). Set each slider:

| Setting | Value | How to set it |
|---------|-------|---------------|
| **Image Size** | 3:4 (Portrait) | Click the aspect ratio picker, choose Portrait, drag to 3:4 |
| **Stylization** | ~750 | Drag the slider to about 75% from the left |
| **Weirdness** | 0 | Leave it at the far left (default) |
| **Variety** | Low | Drag to the left |
| **Model** | Version 7 | Select "Standard" and "V7" if there's a dropdown |
| **Speed** | Fast | Select Fast (uses more GPU minutes but much quicker) |

These settings stay saved between sessions. You only need to set them once.

## Part 4: Generate the 5 MVP Designs

You'll generate 5 designs, one for each element. Start with Fire Dragon — it's the hero/flagship design.

**How Midjourney works:**
- You paste a prompt into the text box and press Enter
- Midjourney generates 4 images in a grid
- You pick the best one and "Upscale" it (make it full resolution)
- If none of the 4 are good enough, click the refresh button to get 4 new ones

### Design 1: FIRE DRAGON (Hero Design — Generate First)

Go to **midjourney.com/imagine** and paste this prompt:

```
Chinese zodiac Dragon, modern ink wash sumi-e art style. Dragon coiled in ascending spiral, head toward viewer, one claw reaching through mist. Majestic, mythical, commanding. Detailed scales in ink wash, flowing whiskers and mane, cloud and mist swirling. burnt orange and molten gold glow, crimson and vermillion ink washes, amber tones, ember particles, heat shimmer. Bold black ink brush strokes on solid black background, dramatic element-colored glow emanating from animal, ink splatter and calligraphic flourishes, high contrast, no text, no words, no letters, no characters, no kanji, no Chinese characters, no writing, no borders, no ground surface, animal floats in void, centered composition, generous negative space, single subject. Wu Xing Five Elements Collection wearable art --sref 6401195053
```

**What to look for in the results:**
- Dragon should be the clear focal point, centered
- Background should be solid black or near-black
- Orange/gold glow should be visible but not overwhelming
- Ink brush texture should be visible on the dragon's body
- No text, letters, or borders anywhere in the image

**Pick the best one:**
1. Hover over your favorite of the 4 images
2. Click **Upscale** (the button that says U1, U2, U3, or U4 depending on which image)
3. Wait for the upscaled version to generate (takes 10-30 seconds)
4. Right-click the upscaled image and click **Save Image As** (or click the download button)
5. Save it somewhere you can find it (like your Desktop or Downloads folder)

If none of the 4 images are good: click the **Re-run** / **Refresh** button to get 4 new ones. You can do this as many times as you want.

## Part 5: Prepare Images for Print

Each downloaded image needs to be placed on a print-ready canvas before uploading to Fourthwall. You can use any image editor — Photoshop, GIMP (free), Photopea (free, browser-based at photopea.com), or Canva.

### For each of the 5 images:

1. **Create a new canvas:** 4500 x 5400 pixels, 300 DPI
2. **Set the background color:** `#0a0a0a` (near-black — matches the t-shirt color)
3. **Place your Midjourney image** in the center of the canvas
4. **Scale it** so it fills most of the width but has some breathing room on the edges
5. **Export as PNG** (not JPG — PNG keeps sharp edges)
6. **Name the file** using the sign name: `fire-dragon.png`, `wood-tiger.png`, `earth-ox.png`, `metal-horse.png`, `water-snake.png`

**Quick option with Photopea (free, no install):**
1. Go to **photopea.com**
2. File > New > Width: 4500, Height: 5400, DPI: 300, Background: #0a0a0a
3. File > Open & Place > select your Midjourney image
4. Drag corners to resize, center it
5. File > Export As > PNG > Save

### What to Look For

- **Minimal and iconic** — should read clearly at small sizes
- **Circular** emblem shape

- **No text, letters, or borders**
- **Black background** (will become transparent in prep)

Pick the best one for each, Upscale, and Save (same as back prints).

### Prepare for Print

**IMPORTANT:** The final image MUST have a **transparent background** — no black square. If a black box is visible on the shirt, it looks like a sticker. The symbol should float directly on the fabric.

**Do NOT use Color Range or Magic Wand** — the artwork is dark ink wash, so color-based selection will destroy the image.

#### Option A: remove.bg (Easiest)

1. Go to **remove.bg**
2. Upload your Midjourney element symbol
3. It auto-removes the background using AI — works even with dark images
4. Download the transparent PNG
5. Open in Photopea: `File > Open`
6. `Image > Canvas Size` — set to 1500 x 1500 pixels (anchor: center) to add breathing room
7. `File > Export As > PNG`

#### Option B: Photopea Select Subject

1. **Open your Midjourney image** in Photopea: `File > Open`
2. Go to `Select > Subject` — Photopea's AI detects the symbol shape
3. **Copy:** `Ctrl+C`
4. **Create the print canvas:** `File > New` — 1500 x 1500 pixels, 300 DPI, **Transparent** background
5. **Paste:** `Ctrl+V`
6. **Scale:** `Ctrl+T`, type `90` in the W field (chain link locked), press Enter
7. **Center:** Move tool (V), use align center/middle buttons in top toolbar
8. **Verify:** Edges should show checkerboard (transparent), NOT black
9. **Export:** `File > Export As > PNG`

#### File Names

`fire-element.png`, `wood-element.png`, `earth-element.png`, `metal-element.png`, `water-element.png`
